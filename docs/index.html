Selectivity Options for Age-Structured Stock Assessments

A comparative framework using RTMB

This manuscript compares selectivity formulations (logistic, double‑logistic, spline‑based, and time‑varying AR1 surfaces) and emphasizes regularization and identifiability trade‑offs.

Key additions

  • Butterworth, Ianelli, and Hilborn (2003): likelihood‑based assessment with temporally structured selectivity changes for Southern Bluefin Tuna.
  • Time‑varying selectivity comparison table: contrasts structured temporal changes, AR1 surfaces, and block approaches.
